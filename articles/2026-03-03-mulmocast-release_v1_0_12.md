# 🎉 MulmoCast v1.0.12 Released!

マルモキャスト バージョン1.0.12をリリースしました！今回のアップデートでは、音声の読み上げ速度を細かく調整できるようになり、新しい画像生成モデルにも対応しました。

## 1. TTS読み上げ速度オプション

![01-01_speech_params_speed_openai](/images/release_v1_0_12_script/01-01_speech_params_speed_openai.png)

1：TTS読み上げ速度オプションを追加しました。OpenAI TTSでは0.25倍速から4.0倍速まで読み上げ速度を調整できます。

![01-02_speech_params_speed_google](/images/release_v1_0_12_script/01-02_speech_params_speed_google.png)

Google Cloud TTSでは0.25倍速から2.0倍速まで調整できます。プロバイダーごとに設定可能な範囲が異なり、それぞれの上限と下限がUIに表示されます。

## 2. Nano Banana 2 画像生成モデル

![02-01_image_model_nano_banana](/images/release_v1_0_12_script/02-01_image_model_nano_banana.png)

2：画像生成モデルにNano Banana 2を追加しました。Googleの新しいモデル、gemini-3.1-flash-image-previewが画像モデル選択から利用可能です。

## その他の改善

その他、軽微な改善を行いました。起動中のアプリに更新通知が届きます。ダウンロードは公式サイトから行えます。

マルモキャスト バージョン1.0.12をお楽しみください！
